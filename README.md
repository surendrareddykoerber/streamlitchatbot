# streamlitchatbot
This repo contains on streamlit chat GPT example project

# Packages
- streamlit
- openai (Create the api key using the link https://platform.openai.com/account/api-keys)

# Installation
- pip install -r requirements.txt

# How to run
- streamlit run app.py
